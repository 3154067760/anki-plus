# Anki-plus 使用说明

本文档介绍 Anki-plus 的日常使用方法：新建卡片、复习、调整复习节奏。

访问地址：
- **本地：** http://localhost:3030
- **服务器：** http://你的服务器IP:3030

---

## 界面概览

顶部导航有三个标签页：

| 标签 | 功能 |
|------|------|
| 新建卡片 | 创建新的记忆卡片 |
| 今日复习 | 复习到期的卡片 |
| 系统设置 | 调整复习节奏、查看统计 |

右上角显示 **待复习** 数量。

---

## 新建卡片

### 正面 / 背面

每张卡片有 **正面**（问题）和 **背面**（答案），均支持：

- **文本** — 在文本框中输入
- **图片** — 三种方式：
  - 聚焦文本框后 `Ctrl + V` 粘贴截图
  - 点击「批量上传图片」选择文件
  - 拖拽图片到虚线区域
- **音频** — 两种方式：
  - 点击「开始录音」/「停止录音」
  - 点击「上传 MP3」选择文件

### 保存

填写内容后点击 **保存卡片**。保存成功后表单自动清空。

> 正面至少需要文本或图片其中一项。

---

## 今日复习

### 复习流程

1. 点击 **今日复习** 标签
2. 阅读卡片 **正面** 内容
3. 点击 **显示答案**
4. 查看 **背面** 内容
5. 根据掌握程度选择评分按钮

### 四档评分

| 按钮 | 含义 | 效果 |
|------|------|------|
| 重来 | 完全不会 | 重置进度，很快再次出现 |
| 困难 | 勉强记得 | 间隔小幅增长 |
| 良好 | 正常掌握 | 间隔正常增长 |
| 简单 | 非常熟练 | 间隔大幅增长 |

每个按钮下方会显示 **下次复习间隔**，例如 `1天`、`6天`、`1.2年`。

### 复习完成

当没有待复习卡片时，页面显示「今日没有待复习的卡片」。

## 系统设置

在 **系统设置** 页面可调整 SM-2 算法的复习节奏参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始间隔（天） | 1 | 新卡片或「重来」后的第一次间隔 |
| 毕业间隔（天） | 1 | 第一次答「良好/简单」后的间隔 |
| 简单加成 | 1.3 | 选「简单」时，间隔乘以此系数 |
| 困难系数 | 0.5 | 选「困难」时，当前间隔乘以此系数 |
| 最大间隔（天） | 365 | 单张卡片的最长复习间隔 |
| 最低易度因子 | 1.3 | 易度因子不会低于此值 |
| 每日新卡片上限 | 20 | 预留参数 |
| 每日复习上限 | 200 | 预留参数 |

修改后点击 **保存设置**，对新复习的卡片立即生效。

### 统计信息

设置页底部显示：

- **总卡片** — 已创建的全部卡片数
- **待复习** — 当前到期的卡片数
- **新卡片** — 尚未复习过的卡片数

## 手机端使用

### 访问

- **服务器：** 浏览器打开 `http://服务器IP:3030`
- **本地 WiFi 测试：** 使用终端打印的局域网地址，如 `http://192.168.x.x:3030`

### 手机操作差异

| 功能 | 电脑 | 手机 |
|------|------|------|
| 粘贴图片 | Ctrl + V | 长按粘贴（部分浏览器支持） |
| 上传图片 | 拖拽 / 选择文件 | 点击「批量上传图片」 |
| 录音 | 浏览器授权麦克风 | 同上，需 HTTPS 或 localhost |
| 复习评分 | 鼠标点击 | 底部固定评分栏，触控操作 |

> 手机录音功能在 HTTP 非 localhost 环境下可能受限，建议使用「上传 MP3」替代。

## 复习节奏建议

### 日常学习（默认参数）

保持默认设置即可，适合大多数场景。

### 考前突击（缩短间隔）

| 参数 | 建议值 |
|------|--------|
| 初始间隔 | 0.5 |
| 毕业间隔 | 1 |
| 最大间隔 | 30 |

### 长期记忆（拉长间隔）

| 参数 | 建议值 |
|------|--------|
| 初始间隔 | 1 |
| 毕业间隔 | 3 |
| 简单加成 | 1.5 |
| 最大间隔 | 365 |

## 数据说明

所有数据存储在服务器/本地的 `data/` 目录：

```
data/
├── anki.db       # 卡片内容、复习进度、设置
└── uploads/      # 上传的图片和音频
```

- 数据不会上传到第三方
- 删除 `data/` 目录会丢失全部数据
- 建议定期备份 `data/` 目录

## 常见问题

### 新建的卡片什么时候出现复习？

新建卡片 **立即进入待复习队列**，切换到「今日复习」即可看到。

### 评分按钮上的天数是什么意思？

表示如果选择该评分，这张卡片 **下次最早出现的时间间隔**。例如「良好 6天」表示 6 天后再复习。

### 换设备后数据还在吗？

数据存在服务器上，任何设备的浏览器访问同一地址即可看到相同数据。本地开发时数据只在你的电脑上。

### 如何清空所有卡片？

停止服务后删除 `data/anki.db`，重启服务会自动创建空数据库：

```bash
pm2 stop anki-plus
rm /var/www/anki_plus/data/anki.db
pm2 start anki-plus
```

> 此操作不可恢复，请先备份。
